This plugin will automatically connect posts to categories depending on content in posts.

== Description ==

= WordPress Automatic Post Category mapper =
This plugin will match content in your posts with categorynames and hook those together if a match happens.
Let's say you have a category named 'iphone' and the posts with the title or content gets added with the string iphone inside.
Then that post will automatically gets connected to that category. This plugin is perfect for all kinds of autoblogs but even newspaper and magazines.
You can choose which keywords not to match in the free version by adding keywords in settings.

> <strong>Be aware before activation</strong><br>
> Make sure you understand the effect of using this plugin before you activate the automatic scheduler. If you check to "delete non matching" the already connected categories that are attached to posts missing thoose keywords will be taken away.

= Do not match =
You can choose which keywords not to be matched by the plugin in the settings page.

= Scheduler with Cron support =
We have added four new cron scheduler time settings. 1, 5, 15 and 30 minute intervals. You can choose this inside settings.

= PRO Version is under development =
In the upcoming PRO version you will be able to color tag matching keywords and even link them to archive pages automatically. Opt-in when you activate to get news about progress.

== Installation ==

Installing "Automatic Post Categories" can be done either by searching for "Automatic Post Categories" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Goto to Woorocks APC Settings under common settings menu and activate.

== Frequently Asked Questions ==

= What is the plugin for? =

Perfect for all kinds of autoblogs, newspapers and magazines.

= Does it work in all langauges? =

Yes, it only matches categorynames with content in your WordPress installation and takes no notice of langauges.

= Does it match all categories? =

Yes, the free version matches all categories that you have in your installation.

= Does it match all posts or just posts in certain categories? =

Yes, in the free version it will match all posts that have the status PUBLISH and will not take in mind in which categories they are.
